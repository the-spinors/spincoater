Description
-----------
Analysis of the refractive index of PDMS using reflectometry. Results are incoherent with literature,
so we used instead the refractive index from Pfund to calculate film thickness.

Descipción
----------
Análisis del índice de refracción del PDMS por reflectometría. Como los resultados fueron
incongruentes con la literatura, usamos el índice de refracción calculado por Pfund para
calcular el grosor de las películas.

Description
-----------
calculate_index:
	Gets refractive index of PDMS measured with a microscope from apparent hegihts (unused).
thickness_analyzer:
	Yields thickness for PDMS films on a per day basis, using reflectometry (transmitance
	vs wavelength of film): https://physicsopenlab.org/2021/09/15/thin-film-reflectometry/
thickness_vs_RPM_analyzer:
	Compiles result of thickness vs RPM from each batch of films onto graphs.

Descripción
-----------
calculate_index:
	Obtiene el índice de refracción del PDMS por método de alturas aparentes con microscopio
	(sin utilizar).
thickness_analyzer:
	Arroja el grosor de cada grupo de películas, usando reflectometría (transmitancia vs
	longitud de onda): https://physicsopenlab.org/2021/09/15/thin-film-reflectometry/
thickness_vs_RPM_analyzer:
	Compila en gráficas los resultados grosor vs RPM para cada grupo de películas.
